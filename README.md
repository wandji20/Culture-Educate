## About

Culture-Educate is a rails App built on the bases of Ruby on Rails Evaluation in Microverse school for remote developers. When signed_in on Culture-Educate, the user can see all created articles classified by categories. Also the signed_in user can create an article, upvote and downvote a voted article.

## Getting Started

To get a local copy up and running follow these simple example steps:

- Fork this project
- Open your terminal
- Clone this project `git clone git@github.com:wandji20/Culture-Educate.git`
- Go to the project folder `cd Culture-Educate`

### Prerequisites

- Ruby: 2.7.2
- Rails: 6.1.3
- Postgres

### Setup

Instal gems with:

```
  bundle
```

Instal yarn and webpacker with:

```
  yarn install
  rails webpacker:install
```

Setup database with:

```
   rails db:create
   rails db:migrate
```

### Drive local host

Start server with:

```
    rails server
```

Open `http://localhost:3000/` in your browser.

### Run tests

- Open terminal
- Run tests `rspec`
